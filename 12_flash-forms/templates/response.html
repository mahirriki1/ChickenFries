GET request:
- uses requests.args['key'] to get the value of the key
- only used to request data and not modify it
- remains in browser history
- less secure due to data being apart of the URL

POST request:
- uses requests.form['key'] to get the value of the key
- used to modify data
- does not remain in browser history
- more secure since parameters aren't stored in browser history
